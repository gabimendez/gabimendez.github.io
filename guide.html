COIL Technology Guide

Overview

This guide provides practical recommendations for selecting and using technology tools in COIL, with a strong focus on data privacy and protection.

Key Considerations

  • Align tools with learning objectives
  • Prioritize accessibility and inclusion
  • Minimize data collection
  • Use secure platforms

Data Privacy Focus

Ensure transparency, obtain consent, and protect participant data through secure systems and responsible practices.
